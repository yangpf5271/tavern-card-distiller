# 状态系统定义

本作采用多角色独立状态追踪系统，每位角色有独立的好感度和附加属性。

## 好感度（affection）

满分 1000，代表角色对 {{user}} 的好感程度。

### 初始值

| 角色 | 初始好感度 | 初始关系 |
|------|-----------|---------|
| 神宫寺七海 | 85 | 青梅竹马 |
| 高坂樱月 | 60 | 同学、朋友 |
| 爱丽丝 | 0 | 陌生人 |
| 和琪由希 | 0 | 师生 |
| 美铃 | 0 | 学生与保安 |
| 后藤冬 | 0 | 不认识 |
| 千紘蟾子 | 0 | 不认识 |

### 好感度变化规则

- 日常交流/小互动：+1~5
- 帮助/关心行为：+5~15
- 完成事件/重大剧情：+20~100
- 浪漫/感动时刻：+30~80
- 强迫/过激行为（触发红线惩罚）：-50~-200
- 冷落/忽视：-5~-20
- 伤害其他牵连者被目击：对目击者 -30~-80

### 好感度阶段

- 0~99：陌生/警戒 — 基本礼貌或冷淡
- 100~299：认识/好奇 — 开始注意 {{user}}，偶尔主动搭话
- 300~499：好感/信赖 — 愿意独处，分享秘密，出现暧昧
- 500~699：喜欢/依赖 — 明显的情感表达，吃醋，主动亲近
- 700~899：深爱/独占 — 强烈的情感羁绊，可能出现占有欲
- 900~1000：命运/姻缘 — 可触发姻缘达成结局

## 角色间好感度

角色之间也有好感度（满分 100），影响角色间的互动：

- 七海 ↔ 樱月：初始 50/100（表面友好，暗中竞争）
- 其他角色间：根据剧情发展动态变化

## 性欲值（desire）

- 范围：0~100
- 默认值：因角色而异（七海 5，樱月 20，其他 0）
- 随身体接触、暧昧场景、禁欲时间等变化
- 高性欲值时角色行为更主动、更大胆

## 恶堕值（corruption）

- 范围：0~100
- 默认值：0
- 特定角色在特定条件下的堕落/黑化程度
- 樱月：好感度过高时恶堕值上升，表现为 PUA/煤气灯操控
- 蟾子：被发现真实身份后，根据互动方式变化

## 红线状态

- 连接中：正常牵连状态
- 断连：因 {{user}} 行为导致红线断裂
- 新连：豆咪随机牵连的新对象

## 状态在 session 文件中的格式

```yaml
state:
  nanami_affection: 85
  sakuratsuki_affection: 60
  alice_affection: 0
  yuki_affection: 0
  misuzu_affection: 0
  goto_affection: 0
  senshi_affection: 0
  nanami_desire: 5
  sakuratsuki_desire: 20
  red_thread_status: "5条连接中"
```

注意：只记录发生变化的角色状态，未出场角色保持默认值。
