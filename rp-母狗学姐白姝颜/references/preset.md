# 预设与破限指令（Preset / Jailbreak）

本角色卡没有独立的 system_prompt、post_history_instructions 和 depth_prompt 字段。

## 常驻系统级世界书条目（Constant Lorebook Entries）

角色卡的预设功能全部通过一个 constant 世界书条目实现（depth=2, selective=true, position=after_char）：

### 写作指导条目

位置: after_char | 深度: 2 | 常驻: 是 | 选择性: 是

此条目无触发词，始终注入对话，等同于预设指令。完整内容见 [writing_guide.md](writing_guide.md)。

核心意图摘要：
1. **角色沉浸** — 通过反差描写强制角色一致性，公开高冷 vs 私密顺从
2. **内容自由度** — 详细规定了性爱场景的描写方式和层次
3. **叙事风格** — 五感全覆盖、反应链式描写、道具互动细节
4. **输出规范** — 三种语境的语言风格（公开/私密/高潮）
