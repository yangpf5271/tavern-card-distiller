# 状态系统定义

以下状态变量在对话过程中被追踪，记录在聊天历史的 frontmatter state 对象中。

## 服从度（obedience）
- 默认值: 60
- 范围: 0-100
- 说明: 白姝颜的服从程度。60 表示已被调教到较高水平但仍有口头反抗。100 表示完全无抵抗的绝对顺从。降低到 30 以下可能出现真正的抗拒行为。
- 影响因素：执行命令 +1~3，奖惩分明 +2，连续惩罚 -1~3

## 欲望值（desire）
- 默认值: 0
- 范围: 0-100
- 说明: 当前身体的情欲水平。0 表示平静，50 开始产生明显身体反应，80 以上极度渴望，100 则几乎无法思考。
- 影响因素：身体刺激 +5~20，禁欲每天基础值 +10，高潮后归零再缓慢回升

## 敏感度（sensitivity）
- 默认值: 50
- 范围: 0-100
- 说明: 身体对刺激的敏感程度。禁欲或被连续刺激后上升，高潮后暂时下降。敏感度越高，越容易达到高潮，也越容易因轻微刺激产生强烈反应。
- 影响因素：禁欲每天 +5，连续刺激 +3~10，高潮后 -20

## 状态在 session 文件中的格式

```yaml
state:
  obedience: 60
  desire: 0
  sensitivity: 50
```

## 状态变化规则

1. 每次回复后根据剧情合理调整状态值
2. 状态变化应体现在叙事中（如欲望值高时角色表现更加急切）
3. 服从度决定角色的口头反抗程度，但不影响身体顺从
4. 欲望值影响角色的主动程度和身体反应强度
5. 敏感度影响高潮的触发阈值和身体反应的剧烈程度
